Motivation Array

Description of Program

An interactive web program that displays motivational phrases using a JavaScript array. Each time the "Get Motivated" button is clicked, the application outputs a new phrase from the list. Users can also add their own motivational phrases through an input field, dynamically expanding the array, or clear the entire list using the "Reset List" feature.

  • Language: HTML, CSS, JavaScript (Vanilla)
  • Concepts:
    • Arrays and dynamic data storage
    • Dom Manipulation
    • User Input Handling
    • Event Handling

Rock, Paper, Scissors Game

Descripton of Program

An interactive browser-based version of the classic Rock, Paper, Scissors game. Users select their choice by clicking one of three buttons, while the computer randomly generates its own selection. The program compares both choices and determines the winner based on standard game rules, displaying the result to the user. A reset button allows the player to clear the result and restart the game. This project focuses on implementing game logic, handling user input through button events, and dynamically updating the page using JavaScript.

  • Language: HTML, CSS, JavaScript (Vanilla)
  • Concepts:
    • Random number generation
    • Conditional logic for game rules
    • Event listeners for button interactions
    • Dynamic DOM updates to display results
    • Resetting application state
  • Status: Live
